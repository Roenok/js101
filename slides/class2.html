Control Flow

if statements

The if Statement

Use if to tell JS which statements to execute, based on a condition.

What if looks like:

if (condition) {
  // statements to execute
}

How if Works

if statements execute only if your condition is true. Example:

var x = 5;

if (x > 0) {
  console.log('x is a positive number!');
}

Operators

Comparison Operators

Use these operators to compare two values for equality, inequality, or difference.

Comparison Operators

  • Compares two operands (items) and returns true or false
  • Operands can be numerial, string, logical, or object values
  • If the operands aren't the same type, JavaScript tries to convert before comparing, except for...
  • Strict comparisons don't attempt conversion

Equality

var myAge = 28;
myAge == 28;

Important!
Don't confuse assignment (=) with equality comparisons (==)!

Strict equality

Does not convert objects!

var myAge = 28;
myAge === 28;

What happens when we do this?

var age = '28';
age === 28;

Inequality

var myAge = 28;
myAge != 28;

Strict inequality

var myAge = 28;
myAge !== 28;

Greater than

var myAge = 28;
myAge > 28;

Less than

var myAge = 28;
myAge < 28;

Greater than or equal

var myAge = 28;
myAge >= 28;

Less than or equal

var myAge = 28;
myAge <= 28;

Operators

Logical

And, or, not.

Logical Operators

  • Typically used with comparison operators
  • Compare two operands and returns true or false
  • Allow you to test several conditions at once!
  • Always read left to right
  • Only continue testing if needed

&&

Returns true if: both statements are true.

var posNum = 4;
var negNum = -2;

True expressions:

posNum > 0 && negNum < 0
4 > 0 && -2 < 0

False expressions:

posNum > 0 && negNum > 0
4 > 0 && -2 > 0

||

Returns true if: either statement is true.

var posNum = 4;
var negNum = -2;

True expressions:

posNum > 0 || negNum > 0
4 > 0 || -2 > 0

False expressions:

posNum > 0 || negNum < 0
4 > 0 || -2 < 0

!

Returns true if: the statement is not true.

var posNum = 4;
var negNum = -2;

True expressions:

!(posNum === negNum)
!(posNum < 0)

False expressions:

!(posNum !== negNum)
!(posNum > 0)

Using mutiple operators

When combining together multiple conditions, use parantheses to group:

var myAge = 28;
if ((myAge >= 0 && myAge < 3)  || myAge > 90) {
  console.log('You\'re not quite in your peak.');
}

Logical operators are typically used with Boolean (logical) values; when they are, they return a Boolean value. However, the && and || operators actually return the value of one of the specified operands, so if these operators are used with non-Boolean values, they may return a non-Boolean value. The logical operators are described on the slides.

It is important to remember that in both scenarios conditions are tested left to right, and only continue testing if needed.

An important lesson: truthy and falsey

Truthy

If you don't use a comparison or logical operator, JavaScript tries to figure out if the value is "truth-y".

var dogsRule = true;
if (dogsRule) {
  console.log("Yay dogs!");
}

Falsey

Values that are "false-y": false, the empty string (""), the number 0, undefined, null.

var name = '';
if (name) {
  console.log("Hello, " + name);
}

var points = 0;
if (points) {
  console.log("You have " + points + " points.");
}

var firstName;
if (firstName) {
  console.log("Your name is " + firstName);
}
